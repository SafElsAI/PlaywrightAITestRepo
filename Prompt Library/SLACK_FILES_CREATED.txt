  ENVIRONMENT VARIABLES
═══════════════════════════════════════════════════════════════════════

REQUIRED:
  SLACK_ENABLED=true|false
  SLACK_METHOD=webhook|bot-token
  SLACK_CHANNEL=#channel-name

WEBHOOK METHOD:
  SLACK_WEBHOOK_URL=https://hooks.slack.com/services/...

BOT TOKEN METHOD:
  SLACK_BOT_TOKEN=xoxb-...

OPTIONAL:
  SLACK_NOTIFY_PASS=false
  SLACK_NOTIFY_FAIL=true
  SLACK_NOTIFY_SKIP=false
  SLACK_UPLOAD_SCREENSHOTS=false
  SLACK_UPLOAD_TRACES=false

═══════════════════════════════════════════════════════════════════════
  BASIC USAGE
═══════════════════════════════════════════════════════════════════════

// Get notifier instance
const notifier = SlackNotifier.getInstance();

// Send test notification
await notifier.notify({
  testName: 'My Test',
  status: 'passed',
  duration: 1000,
  browser: 'chromium',
  timestamp: new Date().toISOString(),
});

// Send summary
await notifier.sendSummary({
  total: 10,
  passed: 9,
  failed: 1,
  skipped: 0,
  duration: 25000,
});

═══════════════════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

No notifications?
  ✓ Check SLACK_ENABLED=true
  ✓ Verify environment variables
  ✓ Check console for errors

"404 Not Found"?
  ✓ Verify webhook URL is correct
  ✓ Check URL starts with https://hooks.slack.com/

"invalid_auth" with bot token?
  ✓ Verify token starts with xoxb-
  ✓ Regenerate token if expired
  ✓ Reinstall app to workspace

"channel_not_found"?
  ✓ Invite bot to channel: /invite @your-bot
  ✓ Use channel name without #
